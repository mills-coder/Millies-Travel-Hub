The internet is being used more and more, with the increase in usage the need for accessibility is increasingly more important. An example of ways in which we can improve accessibility is through alternative text. For an internet user with a visual impairment alternative text is essential, without it the information is inaccessible. This is because they may use a screen reader, which reading the text off a page cannot interpret images. Whilst accessibility helps people with disabilities it can contribute even further, improving user experience and satisfaction. We can increase fonts, making it easier for people with visual impairments but also older people to view the webpage. When implementing changes to boost accessibility, it is best to work on these at the start of a project, to be more efficient and effective [15].

When creating a website, it is important to consider the different filetypes you want to use. There are a number of different filetypes that can be used to display an image, such as PNG, JPG and GIF. Depending on how high you wish the quality to be on your image and the size of the file, you will need to select a specific filetype. JPG is the most commonly used, with small file size and high compatibility. However, the compression of images in JPG can cause small details to be lost increasing each time the image is saved. It also cannot be used with transparent backgrounds and is not easily layered with other elements. In order to increase image quality a PNG filetype can be used, it doesnâ€™t lose any detail when compressed but the file size increases greatly when saved in this format.
